( CONNECT )

THOUGHT TO FORM

이 포트폴리오에서 저는 퍼블리셔로서 구조와 흐름을 선으로 표현했습니다.

이 포트폴리오에 흐르는 선들은,

In this portfolio, I expressed structure and flow through lines — as a publisher. Each line serves as a tool that connects information, design, and user navigation. These lines are my language, and a core element of my work.
The lines that run through this portfolio represent fragments of the web I've built and connected.
